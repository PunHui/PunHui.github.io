在 VS Code 中运行代码会出现以上的错误提示你没有 a 函数
我们直接在浏览器里搜索 a 函数后点进去

function a(a) {
    var d, e, b = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789", c = "";
    for (d = 0; a > d; d += 1)
        e = Math.random() * b.length,
        e = Math.floor(e),
        c += b.charAt(e);
    return c
}

直接将代码复制进来，然后重复以上操作，直到不报错为止

最后工作

在最后添加以下代码

function token(d, e, f, g) {
    d = JSON.stringify(d)
    var tokens = doc(d, e, f, g);
    return tokens;
}

doc 函数就是 d 函数，我修改了函数名，避免冲突

e, f, g 虽然是固定值，但是写在 Python 里会更加好，以便以后修改
注意参数 d 是不固定的!!!

d = JSON.stringify(d) 添上这个函数我们直接在 Python 里写好字典即可，不用自己将字典修改成字符串类型
